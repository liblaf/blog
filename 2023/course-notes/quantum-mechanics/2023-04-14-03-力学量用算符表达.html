3.1 算符的运算规则

(a) 线性算符

A^(c1ψ1+c2ψ2)=c1A^ψ1+c2A^ψ2 \hat{A} (c_1 \psi_1 + c_2 \psi_2) = c_1 \hat{A} \psi_1 + c_2 \hat{A} \psi_2 A^(c1​ψ1​+c2​ψ2​)=c1​A^ψ1​+c2​A^ψ2​

(A^+B^)ψ=A^ψ+B^ψ \pqty{\hat{A} + \hat{B}} \psi = \hat{A} \psi + \hat{B} \psi (A^+B^)ψ=A^ψ+B^ψ

3.3 共同本征函数

3.3.1 不确定度关系的严格证明

ΔA\vdotΔB⩾12\abs\commAB‾ \Delta{A} \vdot \Delta{B} \geqslant \frac{1}{2} \abs{\overline{\comm{A}{B}}} ΔA\vdotΔB⩾21​\abs\commAB​

这就是 任意两个力学量 AAA 与 BBB 在任意量子态下的不确定度 (涨落) 必须满足的关系式, 即不确定度关系 (uncertainty relation).

若两个力学量 AAA 与 BBB 不对易, 则一般来说 ΔA\Delta{A}ΔA 与 ΔB\Delta{B}ΔB 不能同时为零, 即 AAA 与 BBB 不能共同测定 (但注意 \commAB‾=0\overline{\comm{A}{B}} = 0\commAB​=0 的特殊态可能是例外), 或者说 它们不能有共同本征态. 反之, 若两个厄米算符 A^\hat{A}A^ 与 B^\hat{B}B^ 对易, 则可以找出这样的态, 使 ΔA=0\Delta{A} = 0ΔA=0 与 ΔB=0\Delta{B} = 0ΔB=0 同时满足, 即可以找出它们的共同本征态.

3.3.3 对易力学量完全集 (CSCO)

设给定一组量子数 α\alphaα 之后, 就能够确定体系的唯一一个可能状态, 则我们称 (A^1,A^2,⋯ )(\hat{A}_1, \hat{A}_2, \cdots)(A^1​,A^2​,⋯) 构成体系的一组 对易可观测量完全集.

如体系的 Hamilton 量不显含时间 ttt (\pdv∗Ht=0\pdv*{H}{t} = 0\pdv∗Ht=0), 则 HHH 为守恒量. 在此情况下, 如对易力学量完全集中包含有体系的 Hamilton 量, 则完全集中各力学量都是守恒量, 这种完全集又称为 对易守恒量完全集 (CSCCO) 包括 HHH 在内的守恒量完全集的共同本征态, 当然是定态, 所相应的量子数都称为 好量子数.

关于 CSCO, 再做几点说明:

  1. CSCO 是限于 最小集合, 即从集合中抽出任何一个可观测量后, 就不再构成体系的 CSCO. 所以要求 CSCO 中各观测量是 函数独立的.
  2. 一个给定体系的 CSCO 中, 可观测量的数目一般等于体系自由度的数目, 但也可以大于体系自由度的数目.
  3. 一个给定体系往往可以找到多个 CSCO, 或 CSCCO. 在处理具体问题时, 应视其侧重点来进行选择. 一个 CSCCO 的成员的选择, 涉及体系的对称性.

3.3.4 量子力学中力学量用厄米算符表达

量子体系的可观测量 (力学量) 用一个线性厄米算符来描述, 也是 量子力学的一个基本假定, 它们的正确性应该由实验来判定. “量子力学中力学量用相应的线性厄米算符来表达”, 其含义是多方面的.

  1. 在给定状态 ψ\psiψ 之下, 力学量 AAA 的平均值 A‾\overline{A}A 由下式决定

    A‾=(ψ,A^ψ)/(ψ,ψ) \overline{A} = \pqty{\psi, \hat{A} \psi} / \pqty{\psi, \psi} A=(ψ,A^ψ)/(ψ,ψ)

  2. 在实验上观测某力学量 AAA, 它的 可能取值 A′A'A′ 就是算符 A^\hat{A}A^ 的某一个本征值. 由于力学量的观测值总是实数, 所以要求相应的算符为厄米算符.

  3. 力学量之间的关系也通过相应的算符之间的关系反映出来. 例如, 两个力学量 AAA 与 BBB, 在一般情况下, 可以同时具有确定的观测值的必要条件为 \commA^B^=0\comm{\hat{A}}{\hat{B}} = 0\commA^B^=0. 反之, 若 \commA^B^≠0\comm{\hat{A}}{\hat{B}} \neq 0\commA^B^=0, 则一般来说, 力学量 AAA 和 BBB 不能同时具有确定的观测值.
